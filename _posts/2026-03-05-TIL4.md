# c++ 문법
## char 변수
```cpp
int main()
{
    char ch = 'a';
    char str[] = "apple";
    std::cout << str;
    return 0;
}
```

## 함수
```cpp
void HA()
{
    std::cout << "Ha";
    std::cout << "Ha";
    std::cout << "Ha";
}
int main()
{
    std::cout << "Ha";
    HA();
    return 0;
}
```
- HA() : 함수의 이름
- void : 반환값
- main 함수에서 사용할 함수라면, main 함수보다 위쪽에 만든다.

#### 지역변수
- 해당 함수 안에서 만들어진 변수
- 그 안에서만 사용 가능  
  
#### 전역변수
- 함수 밖에서 만들어진 변수
- 프로그램 전체에서 사용가능

## ASCII 코드
- 메모리는 숫자만 저장 가능하다.
- 그 중에서도 2진수만 저장 가능하다.
- 문자를 저장하기 위해 문자를 숫자로 나타내도록 만든 규칙을 ASCII코드라 한다.
- 아스키코드는 128개의 문자로 구성되어있다. 

```cpp
int main()
{
    char ch = 'A';
    // 형변환. 타입 바꾸기
    int iCh = (int)ch;
    std::cout << iCh;
    return 0;
}
```

#### A부터 Z 출력
```cpp
int main()
{
    for (char ch = 'A'; ch <= 'Z'; ch++)
    {
        std::cout << ch;
    }
}
```

#### 대소문자 바꾸기
```cpp
int main()
{
    char ch = 'a';
    char Ach = ch - 32;
}
```

# 오늘의 삽질
## C++에서는 큰 따옴표(")랑 작은 따옴표(')를 완전히 다르게 사용한다.
- 작은 따옴표는 문자(char) 한 글자
- 큰 따옴표는 문자열(string)
- 작은 따옴표에 문자열 넣을 수 없다.

## char 배열 ≠ 문자열
- 문자열에는 자동으로 '\0'(null 문자)이 붙는다.
- 문자열 메모리 예시 : [a][p][p][l][e][\0]
- char 배열
- `char str[4] = {'a', 'b', 'c', 'd'};`
- char 배열 메모리 예시 : [a][b][c][d]
- `std::cout << str;` : 위험
- cout은 str을 문자열이라고 가정하기 때문에 '\0'이 나올때까지 계속 출력한다.
