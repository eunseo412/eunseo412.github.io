### 입출력 함수
``` cpp
std::cin >> a;
```
- console input 약자
- 입력받은 값을 변수(박스)안에 넣어주는 명령어


```cpp
#include <iostream> 
int main()
{
    std::cout << "Hello world!"
    int a = 3;
    int b;
    b = 10;

    std::cin >> a >> b;
    std::cout << a << b << std::endl;
    // 변수에 있는 값을 증가시키고 싶을 때
    a = a + 5;
    a += 5;
    a++;
    a--;
    // 1을 더하거나 빼기
    b = ++a;    // 전위연산자
    b = a++;    // 후위연산자
    return 0;
```

## c++은 객체지향적이라는 말의 의미
예를 들어

c언어에서
`printf("Hello");`  
입출력은 그냥 함수이다.

c++에서는
`std::cout << "Hello";`  
<<도 cout 객체 안에 정의된 기능이다.  
`cout.operator<<("Hello");`

표준 라이브러리 대부분이 객체 기반이다.
