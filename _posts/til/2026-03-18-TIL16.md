# C++
## 문자와 문자열
```cpp
- 문자열은 여러 글자로 구성되어 있기 때문에 char이 아니라 char배열로 저장 가능하다.
- 문자열로 초기화를 할때는  실제 문자열의 문자 개수보다 한 개 더 크게 만들어야한다.
- 문자열 마지막은 '/0'으로 끝나기 때문이다.

int main()
{
    // 문자열 하드코딩
    char str[5] = "ABCD";
    // 문자 하드코딩
    char chs[3] = {'A', 'B', 'C'};

    // 문자열의 길이 구하기
    int len = 0;
    for (int i = 0; i < 5; i++)
    {
        if (str[i] == '\0')
        {
            len = i;
        }
    }

    cout << "문자열의 길이: " << len << endl;
    cout << str << endl;
}
```

#### char 배열 문자와 문자열 출력 차이
- 단순히 cout 변수명을 했을 때 문자열은 제대로 출력되는 반면
- 문자 3개를 넣어준 배열은 한글자씩 출력해야한다.
- \0이 없으니 어디가 끝인지 몰라서 쓰레기값이 출력되기 때문이다.

#### &str[0] 의미
`cout << str << endl;`
`cout << &str[0] << endl;`
- 뒤 두 코드는 동일하다.
- 어차피 [0]부터 할거면 그냥 &str[0]을 문자열 이름 str으로 하자고 약속한거다.



## 2중 for문의 활용
- 일단 대강 로직을 작성하고 디버깅을 해가면서 잘못된 부분을 눈으로 확인하며 로직을 짜주면 더 쉽다.
```cpp
int main()
{
    int vect[3][3] = {};
    int t = 1;
    for (int y = 0; y < 3; y++)
    {
        for (int x = 2 - y; x <= 2; x++)
        {
            vect[y][x] = t;
            t++;
        }
    }
}
```
  

## pointer 를 활용하여 return이 두 개인 효과
- 함수의 return은 한 개이다.
```cpp
int test(int d)
{
    if (d == 3)
    {
        return 1;
    }
    else
    {
        return 0;
    }
}
```
  
- return을 여러 개 하고 싶은 경우에 아래와 같이 사용할 수 있다.
- main함수에서 kfc함수를 사용함으로서 값 두개가 전달되기 때문에 kfc함수는 return이 두개인 상황과 유사하다고 볼 수 있다.
  
```cpp
void kfc(int *a, int *b)
{
    *a = 100;
    *b = 200;
}

int main()
{
    int h = 0;
    int k = 0;
    kfc(&h, &k);
}
```

## 함수에 배열을 인자로 전달하기
- pointer 변수는 *을 찍어서 해당 주소값의 변수위치로 이동도 할 수 있고
- 또한 배열처럼 사용도 가능하다!!
  
```cpp
void ArrayTest(int *pArr)
{
    cout << *pArr << endl;
    // *pArr는 int형 포인터이다.
    // *(pArr + 1)에서 1은 사실 size of (int)이다. 즉 4바이트!
    cout << *(pArr + 1) << endl;
    cout << *(pArr + 2) << endl;
    // 그렇지만 너무 불편한 방법! 데이터타입확인, 바이트 연산이 필요하기 때문
    // 위 코드는 안쓴다.

    // 주소를 넘기고 pointer 변수를 배열처럼 사용할 수 있다.
    cout << pArr[0];
    cout << pArr[1];
    cout << pArr[2];
    cout << pArr[3];
}

int main()
{
    int arr[5] = {1, 2, 3, 4, 5};
    ArrayTest(arr);
    // 이게 되는 이유는? arr가 &arr[0]를 의미하기 때문이다. arr 0번째의 주소를 넘겨준거다.
}
```
  
## idx 백업변수
```cpp
int main()
{
    int arr[5] = {1, 2, 3, 4, 5};

    int max = 0;
    int maxIdx = -1;
    for (int i = 0; i < 5; i++)
    {
        if (arr[i] > max)
        {
            max = arr[i];
            maxIdx = i;
        }
    }
    cout << "max: " << max << "maxIdx: " << maxIdx << endl;
}
```
## 버블 정렬
```cpp
int main()
{
    int arr[6] = {5, 3, 4, 2, 6, 1};

    // 버블 정렬
    for (size_t y = 0; y < 6; y++)
    {
        for (size_t x = 0; x < 5 - y; x++)
        {
            if (arr[x] > arr[x + 1])
            {
                int temp = arr[x];
                arr[x] = arr[x + 1];
                arr[x + 1] = temp;
            }
        }
    }
}
```
