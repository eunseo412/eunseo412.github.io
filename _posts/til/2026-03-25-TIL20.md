# AI Backend ssh 접속  
  
`./connect.sh $PORT ./id_container`
`$PORT 자리에는 서버 열었을때 정해지는 port번호 작성

그 전에 ./connect.sh에
chmod +x ./connect.sh 해주기

- ssh 접속시 옵션 추가
`ssh -Y -i ./id_container -p 53561 work@127.0.0.1`

최종 정리
### 로컬에서 
`socat TCP-LISTEN:6000,reuseaddr,fork UNIX-CLIENT:\"$DISPLAY\"`
- 서버에서 보낸 신호를 Mac의 XQuartz 유닉스 소켓으로 중계해줘야 한다. 
- 이 창은 계속 켜두어야 한다.(로컬 터미널에서 깜빡깜빡)
  
### 또다른 로컬창에서
`cd Document/2026-1/deep_learning`
`ssh -Y -i ./id_container -p 53561 -R 6000:127.0.0.1:6000 work@127.0.0.1`
- -R 6000:127.0.0.1:6000: 서버의 6000번 포트(X11 기본 포트)를 내 Mac의 6000번 포트로 바로 쏴주는 설정이다.

# C++ 
## 클래스와 구조체의 차이
- 구조체의 디폴트 접근제한 지정자는 public
- 클래스의 디폴트 접근제한 지정자는 private

  
### 구조체
```cpp
struct People
{
    // public: (default)
    int age;
    int height;
    char name[256];
};
```

### 클래스
```cpp
class cPeople
{
    // 접근 제한 지정자
    // private: (default)
    int age;
    int height;
    char name[256];
};
```
  
### 클래스 데이터 변경
```cpp
void printPeople(People* p1)
{
    //p1이라는 메모리 공간이 새로 생긴다.
    // p1의 데이터가 바뀌는거다.
    // p1.age += 1;
    (*p1).age += 1;
}
```
  
### 클래스와 구조체 생성
```cpp
int main()
{
    People Eunseo;
    Eunseo.age = 3;
    Eunseo.height = 120;
    // 문자열 복사
    strcpy_s(/*&Eunseo.name[0]*/&Eunseo.name, "SeoEunseo")
    // Eunseo.name = "SeoEunseo" 는 당연히 불가.
    // 클래스 생성하면서 배열이 만들어져있기 때문이다.
    // 이런 형태는 배열 초기화 시 사용한다.

    cPeople Minseo;
    // Minseo.age
    //private이므로 Minseo.~ 에 접근 불가하다.
}
```
  
- public과 private을 지정할 수 있다.
```cpp
class cPeople
{
public:
    int age;
    int height;

private:
    char name[256];
};
```
