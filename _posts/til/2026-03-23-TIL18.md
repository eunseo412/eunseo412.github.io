## destructor

- ~class명 : destructor
- 안 쓰는 경우메모리를 비워서 그공간에 다른 데이터를 넣을 수 있도록

- destructor가 실행되는 순서는 instructor가 실행되는 순서의 반대이다.

## 2차원 배열의 max값 구하기
```cpp
int main()
{
    int arr[4][4] =
        {
            4, 1, 3, 14,
            5, 32, 1, 5,
            2, 7, 49, 2,
            50, 11, 3, 29};

    int max = 0;
    for (int y = 0; y < 4; y++)
    {
        for (int i = 0; i < 4; i++)
        {
            if (arr[0][i] > max)
            {
                max = arr[y][i];
            }
        }
    }
}
```

## 두 배열 비교하기

{

    int numArrA[5] = {1, 23, 4, 5, 6};

    int flag = 0;
    for (int i = 0; i < 5; i++)
    {

        {
            flag = 1;
