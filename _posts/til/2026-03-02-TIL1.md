## 2. c++ 파일 실행 명령어
### 컴파일
`clang++ 파일이름.cpp -std=c++17 -o 실행파일이름`
- -g : 디버그 정보 포함
- -Wall : 경고 보기
- -o : 컴파일된 파일명 지정
- -std=c++17 : c++문법 버전 지정

### 컴파일된 파일을 실행
./실행파일이름
