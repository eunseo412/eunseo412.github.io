## 2차원 배열 연습
- 숫자 대입되는 순서 생각하며 반복문 짜기
- 머리로만 하지 말고 각 index 작성한 다음 생각하기
  
```cpp
int main()
{
    int arr[3][3] = {};
    int num = 0;
    for (int y = 0; y < 3; y++)
    {
        for (int x = 0; x < 3; x++)
        {
            arr[y][x] = num;
            num++;
        }
    }
}

int main()
{
    int arr[3][3] = {};
    int num = 0;
    for (int x = 2; x >= 0; x--)
    {
        for (int y = 0; y < 3; y++)
        {
            arr[y][x] = num;
            num++;
        }
    }
}
```

## reinforcement learning 
- key concept : trial and error
  - 자전거를 배우듯 시행착오로 학습  
- 기본 개념
  - reward를 최대화하기 위해 environment와 상호작용
  - state(or observation), action, reward
- 강화학습을 사용하는 경우
  - action이 environment에 영향을 미치는 경우 강화학습으로 문제를 풀 수 있다.
  - 출력결과가 입력에 영향을 미치는 경우.
  - 카메라 노출이 크게 다른 경우, 로봇 다리 하나가 부러진 경우 등 특수한 환경에서도 잘 작동하도록 robust control intelligence
