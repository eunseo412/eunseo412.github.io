# c++ 문법

## if문의 다중비교
```cpp
int main()
{
    int x;
    std::cin >> x;

    if ((1 <= x && x <= 3) || (x >= 7 && x <= 12))
    {
        std::cout << "참입니다." << std::endl;
    }
    return 0;
}
```


## for반복문

```cpp
int main()
{
    int max = 0;
    std::cin >> max;

    // 초기값; 조건식; 증감식
    for (int i = 0; i < max; i++)
    {
        if (i = 5)
        { 
            std::cout<< "5번째입니다.";
        }

        std::cout << "안녕하세요!\n";
    }
    return 0;
}
```

## 배열
```cpp
int main()
{
    // 변수 4개를 가지는 배열 만들기
    int arr[4];
    arr[0] = 1;
    arr[1] = 2;
    std::cout << arr[0] << std::endl;
    arr[1] = 10;
    std::cout << arr[1] << std::endl;

    // 배열 하드코딩
    // 초기화방식
    int arr2[4] = {1, 2, 3, 4};

    arr2[0] = 10;
    arr2[1] = 20;
    arr2[2] = 30;
    arr2[3] = 40;

    // {}: 배열 0으로 초기화
    int vect[7] = {};

    return 0;
}
```


## 배열 활용

```cpp
int main()
{
    // 반복문을 이용해 배열에 값 넣기
    for (int i = 0; i < 7; i++)
    {
        vect[i] = 7;
    }

    // sum 변수로 배열의 값의 합 구하기
    int arr3[4] = {1, 2, 3, 4};
    int sum = 0;

    for (int i = 0; i < 4; i++)
    {
        sum += arr3[i];
    }
    std::cout << sum << std::endl;
    return 0;
}
```

#### num 변수를 활용하는 경우
- 연산이 단순해짐
- 메모리는 더 차지
```cpp
int main()
{
    int arr[4] = {};
    int num = 27;
    for (int i = 0; i < 4, i++)
    {
        arr[i] = num;
        num += 2;
    }
}
```
#### 변수를 활용하지 않는 경우
- 연산이 복잡해짐
- 메모리는 덜 차지
```cpp
int main()
{
    int arr[4] = {};
    for (int i = 0; i < 4; i++)
    {
        arr[i] = 27 + (i * 2);
    }
}
```

# 오늘의 삽질
## int main()으로 시작하는 이유?
- c++ 프로그램을 실행하면 운영체제가 내부적으로 main 함수의 주소를 찾아서 호출한다.
- int를 붙이는 이유는 메인 함수가 프로그램 종료 시 return값을 정수형 으로 반환 받겠다는 의미이다.
- 0이 아닌 값이 반환되면 에러가 났다는 의미이다.
  
#### main()함수가 시작점은 아니다.
- main()함수보다 먼저 실행되는 것도 있다.
- 전역 객체 생성자
- static 전역 변수 초기화
- 특수 attribute 함수

## return0; 으로 끝나는 이유?
- 운영체제에게 0을 반환했다는 것이 성공적으로 프로그램을 수행했다는 것을 의미함
- 그냥 약속임
