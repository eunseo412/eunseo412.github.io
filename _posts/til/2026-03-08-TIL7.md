# c++ 문법

## 2차원 배열
```cpp
int main()
{
    int num = 0;
    int arr2D[3][5] = {};

    for (int y = 0; y < 3; y++)
    {
        for (int x = 0; x < 5; x++)
        {
            arr2D[y][x] = num;
            num++;
        }
    }

    return 0;
}
```
## break문
- 가장 가까운 반복문을 빠져나오는 명령어
```cpp
int main()
{
    int arr[5] = {1, 2, 3, 4, 5};
    for (int i = 0; i < 5; i++)
    {
        if (arr[i] == 3)
        {
            std::cout << "3이 존재한다" << std::endl;
            break;
        }
    }
}
```

## while문
- 조건이 참인동안 소스코드를 끊없이 반복하는 문법
- for문은 시작과 끝을 파악하기 쉬운 구조의 문법으로 구성된 반면
- while문은 종료해야 할 조건을 적어주거나 변수를 이용해서 카운트를 하여 빠져나가야한다.
- 특정 조건이 명시되어있지 않은 경우 while문을 사용한다.

```cpp
int main()
{
    int a = 0;
    while (a < 2)
    {
        std::cout << "a";
        a++;
    }
}
```
  
for문와 while문의 비교
```cpp
int main()
{
    int i = 0;
    while (i < 5)
    {
        std::cout << i << std::endl;
        i++;
    }

    for (int i = 0; i < 5; i++)
    {
        std::cout << i << std::endl;
    }
}
```
