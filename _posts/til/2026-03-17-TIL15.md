# 딥러닝

### keyword
- learning rate
- regularization
  - regularization 없으면 특정 neuron만 기여하게 됨. small number of strong weight 기여. biased result 가능성 높음
  - regularization 있으면 weight가 분배. 모든 뉴런들이 output 출력 위해 기여. 다수의 법칙. 


## Perceptron
- layer가 한 개인 neurol network
- hidden layer가 없음
- linear classifier
- n차원에서 linear boundary만들려면 n-1차원의 boundary(?)필요


## Perceptron algorithm
### step1
- params
  - weight 
  - learning rate
- inputs
  - bias위한 1 포함

- > decision boundary 결정됨

### step2
- grond point = 정답지
- 정답지에 어긋나는 포인트 하나 잡기

### step3
- parameter(w) update
- 그 point가 사실 위에 있어야하는지 아래에 있어야하는지에 따라 d값 결정
- L(cost)를 최소화하는 방향으로 w를 수정.
- 따라서 d값은 정답과 오답값의 차이임.(?)
- learning rate를 d에 곱한 값으로 weight를 변화시킴.

## flag 활용
```cpp
int main()
{
    int arr[5] = {1, 2, 3, 4, 5};

    int flag = 0;
    for (int i = 0; i < 5; i++)
    {
        if (arr[i] == 2)
        {
            flag = 1;
        }
    }
    if (flag == 1)
    {
        cout << "존재" << endl;
    }
    else
    {
        cout << "존재 X" << endl;
    }
}
```

## 최댓값 찾기
```cpp
int main()
{
    int max = 0;
    int arr[5] = {1, 2, 3, 2, 5};

    for (int i = 0; i < 5; i++)
    {
        if (arr[i] > max)
        {
            max = arr[i];
        }
    }
    cout << max << endl;
}
```
참고: a, b, c 등 문자도 사실 아스키코드이기 때문에 똑같이 최댓값 검색이 가능하다.  
  
## 다른 배열에 값 옮기기
```cpp
int main()
{
    int arr[5] = {1, 2, 3, 2, 5};
    int temp[5] = {};
    int idx = 0;

    for (int i = 0; i < 5; i++)
    {
        if (arr[i] < 3)
        {
            temp[idx] = arr[i];
            idx++;
        }
    }
    return 0;
}
```
