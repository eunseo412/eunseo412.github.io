# Perceptron
binary output을 낼 때 사용한다.
perceptron을 적용할 때 전제 조건으로 data가 linearly seperable 해야한다.

target function f를 찾는 것은 불가능하므로
최적의 hypothesis(decision boundary) g를 찾는다.

어떤 threshold를 기준으로 예 아니오로 답해야하므로
sign function을 사용한다. 넘으면 1, 넘지 못하면 -1

sign(sum(w * x)-threshold)
sign(sum(w * x) + bias)로 표현

특정 x를 sign function에 넣었을 때 정답값(y. ground truth)는 정해져 있다.
그런데! ground truth랑 다른 point를 발견하면?
가설(decision boundary)를 수정해야한다!
hypothesis 수정 방법 : weight 수정

𝐰′= 𝐰 + 𝜂𝑑 ∙ 𝐱

𝜂(nu): learning rate
d: 1 또는 -1
point가 선 위에 있어야하는데 아래에 있는 상황이면 d = 1
point가 선 아래에 있어야하는데 위에 있는 상황이면 d = -1
