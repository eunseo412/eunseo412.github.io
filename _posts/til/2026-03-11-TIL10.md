# C++

## 메모리와 변수형
- int 저장 위해서는 메모리 4바이트 필요
- bool 저장 위해서는 메모리 1바이트 필요

## 주소연산자
- & : 주소연산자
- &변수명 : 변수가 저장된 메모리의 주소값
  
```cpp
int main()
{
    int a = 0;
    bool b = true;

    cout << &a << endl;
    cout << &b;
}
```
주소값 예시
`0x16d0eabbc`
저장된 메모리 공간 시작점의 주소값을 출력 (16진수로 표현)

## pointer
포인터 선언

```cpp
int main()
{
    bool silly = true;
    bool *p;
    p = &silly;
    // bool *p = &silly; 와 동일
    cout << *p;
}
```
- 포인터도 타입을 갖는다.
- 포인터의 타입은 어떤 타입의 변수를 point하는지 말해준다.
- point 하는 데이터가 얼마나 큰지(시작점으로부터 몇 바이트를 봐야할지) 알 수 있다.
- *연산자 (ex)*p: p라는 변수에 저장된 주소를 가져와서, 그 공간에 있는 데이터를 가져온다. 다른 공간에 같은 크기의 같은 데이터를 저장한다.(?)

```cpp
int main()
{
    bool silly = true;
    bool *p;
    p = &silly;

    cout << &silly << endl; // silly address
    cout << p << endl;      // silly address
    cout << *p << endl;     // true
    cout << silly << endl;  // true
}
```

## 포인터 유무 비교(데이터 바꾸기)
### 포인터를 사용하지 않은 경우
```cpp
void set(int x)
{
    x = 5;
}
int main()
{
    int a = 1;
    set(a);
    cout << a; //1
}
```
- x는 local variable이다.
- a의 값에는 영향이 없다.


### 포인터를 사용한 경우
```cpp
void set(int *x)
{
    *x = 5;
}
int main()
{
    int a = 1;
    set(&a);
    cout << a;
}
```
- 변수 x는 a의 주소값을 갖는다.
- 포인터 x(*x)는 1에서 5로 변한다.

## common pointer mistake
```cpp
int main()
{
    int *p = nullptr; // 아무것도 pointing하고 있지 않다고 선언해줘야 한다.
    *p = 42;
    cout << *p;
}
```


## &로 받아서 값 바꾸기 - 처음에 헷갈리는 부분!
```cpp
void foo(int &b)
{
    b = 5;
}
int main()
{
    int a = 3;
    foo(a);

    cout << a << endl;
}
```
