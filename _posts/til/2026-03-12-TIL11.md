# C++
(0312 자료구조 수업 내용)

- 일반적인 pointer 지정하기

```cpp
int main()
{
    int x = 10;
    int *p = &x;

    *p = 30;
    cout << x << endl;
}  
  
// 또는
  
void set(int *v)    //*v = &x
{
    *v = 10;
}
int main()
{
    int x = 1;
    set(&x);
    cout << x << endl;
}
```

---
- x를 &r로 받았기 때문에 r은 x와 같다.
```cpp
int main()
{
    int x = 5;
    int &r = x;
    r = 20;
    cout << x << endl;
}
```


```cpp
int main()
{
    int x = 5;
    int y = 7;

    int &r = x;     // r이랑 x랑 같음.  r이 바뀌면 x도 바뀜
    r = y;  // 사실 x에 assign한거임
    cout << r << " " << x << endl;
}

int main()
{
    int x = 5;
    int y = 7;

    int *p = &x;
    p = &y;
    *p = 10;
    cout << x << "" << y << endl;
}
```

## function과 method의 차이
- method는 class에 정의된 function이다.
- 특정 객체에서만 method를 사용할 수 있다.


### constructor
- 클래스의 인스턴스가 생기는 시점에 실행되는 method
- python 클래스의 `__init__`과 동일한 역할
- 클래스명과 constructor이름이 같음
- 사용자가 만들지 않아도, 클래스 만들면 (안보이더라도) 기본 constructor 생성됨

### destructor
- 클래스 사라질 때 실행되는 method
