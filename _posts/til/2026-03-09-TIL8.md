# C++

## swap
- 두 개의 변수를 서로 교환하는 작업
- 임시로 변수를 만든다.
```cpp
int main()
{
    int a = 10;
    int b = 5;
    int temp = a;
    a = b;
    b = temp;
}
```
  
## 구조체
- 기존의 여러가지 변수 타입들을 묶어서 하나의 새로운 타입을 정의하는 문법
- 변수들을 묶은 형태로 그룹화 하여 특정 상황에서 소스코드를 간결하게 표현할 수 있다.
  
```cpp
struct ABC
{
    int num;
    char ch;
};

int main()
{
    ABC abc;
    abc.num = 100;
    abc.ch = 'A';
    cout << abc.num << " " << abc.ch << endl;
}
```
  
## Call by Value
- 전달인자 부분의 값은 복사가 되어 넘어가는 것

```cpp
void test(int n)    // n은 전달인자이다.
{
    for (int i = 0; i < n; i++)
    {
        cout << "test";
    }

}

int main()
{
    if (true)
    {
        int g = 1;
        test(g);    // n은 test 함수내의 지역변수이다. g의 값을 복사해서 지역변수가 새로 만들어진다.
    }
    else
    {
        test(3);
    }
}
```
